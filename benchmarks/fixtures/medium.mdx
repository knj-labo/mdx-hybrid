## Introduction

MDX is a powerful format that combines Markdown with JSX, allowing you to write content with embedded React components.

### Why MDX?

1. **Component Integration**: Seamlessly use React components in your content
2. **Markdown Simplicity**: Write content using familiar Markdown syntax
3. **Dynamic Content**: Create interactive documentation and blog posts
4. **Type Safety**: Full TypeScript support when configured properly

## Code Examples

Here's a simple example of using MDX:

```jsx
import { useState } from 'react'

export function Counter() {
  const [count, setCount] = useState(0)
  
  return (
    <div>
      <p>Count: {count}</p>
      <button onClick={() => setCount(count + 1)}>
        Increment
      </button>
    </div>
  )
}
```
