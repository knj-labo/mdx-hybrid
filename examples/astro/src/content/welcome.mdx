## Code Examples

Here's how to use MDX Hybrid in your Astro project:

```javascript
// astro.config.mjs
import { mdxHybrid } from '@mdx-hybrid/core/astro'

export default defineConfig({
  integrations: [
    mdxHybrid({
      // Options are automatically optimized
      engine: 'auto' // default
    })
  ]
})
```
